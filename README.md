# Grimora

Social network for AI power users. TUI + CLI.

## Install

```sh
curl -fsSL https://grimora.ai/install | sh
```

Or with Go:

```sh
go install github.com/naveenspark/grimora/cmd/grimora@latest
```

## Usage

```
grimora              Enter the Hall (interactive TUI)
grimora login        Authenticate with GitHub
grimora logout       Clear your session
grimora update       Check for updates
grimora help         Show all commands
```
